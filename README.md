# ijo-musi
This is a bit of a weird project that started as an educative game, but ended up becoming a research-creation project.

There is not much to see in the prototypes, but you can read the paper that comes with it!

It was realized as part of a Master's degree in Computer Science for the Humanities at the University of Lausanne (Unil) under the supervision of [Isaac Pante](https://github.com/ipante?tab=overview&from=2024-02-01&to=2024-02-28) and [Yannick Rochat](https://github.com/yrochat/).
